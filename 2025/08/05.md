## わかったこと
- do-while文
    - do-while文はwhile文とほとんど同じだが実行順序が異なる。
```javascript
do {
    実行する文;
} while (条件式);
```
 **do-while文の実行フロー**
1. 実行する文を実行
2. 条件式 の評価結果がtrueなら次のステップへ、falseなら終了
3. ステップ1へ戻る
- Array.prototype.at
```javascript
const array = ["a", "b", "c"];
//
console.log(array.at(0)); // => "a"
console.log(array.at(1)); // => "b"
// 後ろから1つ目の要素にアクセス
console.log(array.at(-1)); // => "c"
// -1は、次のように書いた場合と同じ結果
console.log(array[array.length - 1]); // => "c"
```
